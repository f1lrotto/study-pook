Timeline (Latest First)

#1Styling

Rethemed app to pastel light-blue palette

Global tokens + components updated (panels, badges, pills, meters, wheel, hero blocks, editor chrome) for a softer blue identity.

#2EditingRich text

Imported text notes now edit in rich text mode

Changed imported note editing from plain textarea to rich editor, preserving structured roles (paragraph/list/subheading) and bold segments on save.

#3Scope

Removed edit controls from /courses

Courses page is now read-only overview/navigation; editing remains on /theme only.

#4NotesFallback

Manual notes editor for themes with no imported notes

When a theme has no imported notes, user can create rich notes with formatting, lists, and pasted/inserted images.

#5UX

Changed imported-note editing to whole-block flow

Instead of editing each line separately, note blocks can be edited as one coherent unit.

#6Data behaviorImport overwrite

Clarified source-of-truth and re-import behavior

PDF/DOCX imports replace imported note blocks by external keys; manual notes remain stored and are not overwritten by import.

#7Layout

Increased overall app max width by ~20%

Wider reading canvas to reduce cramped layout on desktop.

#8Dashboard

Removed “Vyžaduje pozornosť” section

Dropped low-value dashboard block to simplify the main page.

#9Study flow

Added “Next theme” CTA on theme page

After finishing a theme, user can jump directly to the next one in course order.

#10Layout

Made sidebar collapsible on /theme

Subthemes + progress panel can be tucked away for distraction-free note study.

#11Overview

Show missing-notes indicator on /courses

Each course now signals how many themes still do not have notes.

#12ScopeMVP trim

Removed flashcards feature for v1

Dropped generator/review/study card flow due to quality concerns to keep MVP focused.

#13Visual polish

Improved emphasis + readability of key states

Adjusted confidence/status visuals and typography hierarchy so progress signals are easier to scan.

#14Typography

Upsized and normalized section/course headers

Fixed too-small headers on /theme, /courses, and main sections for stronger visual hierarchy.

#15Import qualityDOCX structure

Preserved DOCX structure better (bullets/subheadings/images)

Adjusted parsing/rendering so note flow reflects original document structure more closely, including inline images without OCR.
